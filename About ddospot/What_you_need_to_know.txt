Features of This Honeypot:

    Multi-protocol support (HTTP, SSH, DNS, NTP)
    Attack logging with timestamps and details
    Rate-based DDoS detection
    IP blacklisting with automatic cleanup
    Attack summary generation
    Extensible design for adding more protocols

Extending the Honeypot:

    Add more services by calling start_service()
    Adjust detection thresholds in the config
    Integrate with firewall to automatically block blacklisted IPs
    Add database storage for persistent attack data
    Implement machine learning for better attack classification

DDoS Detection

    Rate limiting + auto-blacklisting

Logging	JSON 

    logs + summary file

Easy Start	

    python3 honeypot.py

Test Compatible	

    Works with test_attacks.py

Extensible	

    Add services via services_config
