# DDoSPoT

A Python-based DDoS honeypot designed to emulate vulnerable UDP services
and collect telemetry on amplification attacks.

## Features
- UDP service emulation
- Telemetry logging
- Replay and ML modules
- Docker support

## Installation
```bash
pip install -r requirements.txt
python main.py
```
